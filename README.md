# SyntaxWear

SyntaxWear é um projeto de e-commerce de página única para uma loja de calçados fictícia. O site apresenta um design limpo e moderno, com foco na experiência do usuário.

## Tecnologias Utilizadas

* HTML5
* CSS3

## Funcionalidades

* Design responsivo para uma visualização otimizada em todos os dispositivos.
* Seções de produtos em destaque, categorias e novidades.
* Cabeçalho e rodapé navegáveis.

## Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
